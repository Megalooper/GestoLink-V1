# Sistema de Control de Hardware por Gestos 🖐️🤖

Este proyecto integra **Visión Artificial** y **Sistemas Embebidos** para controlar actuadores (LEDs) de forma remota mediante gestos de la mano. Desarrollado como prototipo para el departamento de Sistemas.

## 🚀 Características
- Detección en tiempo real de 21 puntos clave de la mano.
- Control individual de 5 canales de salida vía Serial.
- Interfaz visual con retroalimentación binaria.

## 🛠️ Requisitos
- **Python 3.10+**
- **Arduino Uno** o compatible.
- Webcam USB.
